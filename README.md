# Dapp Chat

Chat-oriented web app project currently scaffolded as a Svelte application.

## Tech Stack

- Svelte
- JavaScript
- Vercel deployment target

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - start development server
- `npm run build` - create production build
- `npm run start` - run built app
